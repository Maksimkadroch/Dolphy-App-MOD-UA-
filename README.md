# Dolphy

Android multi-tool designed for wireless protocol research and hardware emulation.

## Key Features

- Protocol Research: Support for NFC, BLE, and IR protocols.
- Hardware Emulation: Simulates specific hardware functions directly on Android.
- No Root Required: Works completely on stock firmware without root access.

## Project Status

This repository serves as the official landing page for the project. The source code is currently private.
